# Wordpress docker

Extended php-apache Docker image.

## features

- node 7.x
- deploy user with sudor
- Wordpress 4.5.3 DE_de
- Wordpress plugins
  - wp-nested-pages:1.5.4
  - timber-library:1.0.4
  - regenerate-thumbnails:latest
